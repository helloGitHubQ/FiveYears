## NIO

**阻塞和非阻塞：**

- 阻塞

调用方等待操作完成后返回

- 非阻塞

调用方无需等待操作完成

**同步和异步：**

- 同步

调用者发起请求，等待操作完成再返回

- 异步

调用者发起请求，不需要等待操作完成
